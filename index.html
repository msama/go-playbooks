go-playbooks

Various hopefully useful playbooks for solving common problems with go.

Each playbook starts form a GO library and explains how to use it to solve a practical problem.

Sitemap

.
├── encoding
│   └── json
│       ├── basetypes-marshalling
│       └── custom-marshaller

    └── http
        ├── handling-service-termination
        ├── redirecting-to-https
        │   ├── aws-ebs-nginx
        │   └── handle-redirect
        └── request-caching

encoding/json

  • Basetype marshalling: how to marshal basetypes and how to handle missing fields.
  • Custom marshalling: how to implement custom marshalling and unmarshalling logic.

net/http

  • Handling service termination: how to gracefully terminate http handling.
  • Serving https with AWS EBS: how to serve https requests with AWS EBS.
  • Handling https redirect: how to programmatically redirecting from http to https.
  • Http request caching: how to cache http requests.

About

GO is a great language. I am writing these playbooks to help other developers to practice it.

How to contribute

Pull requests are welcomed. Also if you want to request a Playbook just open a ticket.
